# Compare the Triplets

## Problem
Compare Alice's and Bob's ratings in three categories.

## Approach
- Compare each element one by one.
- If Alice's score is greater, Alice gets 1 point.
- If Bob's score is greater, Bob gets 1 point.
- Ignore equal values.

## Time Complexity
O(1)

## Space Complexity
O(1)

## Language
C++
