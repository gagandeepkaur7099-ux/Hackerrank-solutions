# Plus Minus

## Problem
Given an array of integers, calculate the ratios of:
- Positive numbers
- Negative numbers
- Zeros

Print each ratio on a new line with exactly 6 decimal places.

## Example

Input:
```
5
1 1 0 -1 -1
```

Output:
```
0.400000
0.400000
0.200000
```

## Approach

1. Traverse the array once.
2. Count positive, negative and zero elements.
3. Divide each count by the total number of elements.
4. Print the result with 6 decimal places.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Language

C++
