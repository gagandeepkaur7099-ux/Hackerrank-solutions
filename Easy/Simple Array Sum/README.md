# Simple Array Sum

## Problem

Given an array of integers, find the sum of all its elements.

## Language

C++

## Approach

- Read the size of the array.
- Store all elements in a vector.
- Traverse the vector using a loop.
- Add each element to the running sum.
- Return and print the final sum.

## Algorithm

1. Read the number of elements.
2. Store all elements in a vector.
3. Initialize `sum = 0`.
4. Traverse the array.
5. Add every element to `sum`.
6. Return the total sum.

## Time Complexity

O(n)

## Space Complexity

O(1)
