# Solve Me First

## Problem

Read two integers and print their sum.

## Language

C++

## Approach

- Read two integers from input.
- Pass them to the `solveMeFirst()` function.
- Return the sum.
- Print the result.

## Algorithm

1. Read two integers.
2. Call `solveMeFirst(a, b)`.
3. Add both numbers.
4. Return the sum.
5. Print the answer.

## Time Complexity

O(1)

## Space Complexity

O(1)
